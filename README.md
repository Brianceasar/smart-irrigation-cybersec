# Cybersecurity Attack and Defense Analysis of a Smart Irrigation System

This project simulates a smart irrigation IoT system using MQTT and demonstrates real-world cyberattacks and defense mechanisms.

Components:
- Simulated IoT device (Python)
- MQTT Broker (Mosquitto on Ubuntu)
- Simulated dashboard
- Kali Linux attacker

Attacks demonstrated:
- Unauthorized access
- Command injection
- Replay attack
- DoS
- Packet sniffing

Security mechanisms implemented:
- Authentication
- TLS encryption
- Message integrity validation
- Firewall configuration
